Step 2: Your First Python Program

print("Hello, world!")
Explanation: This uses the built-in print() function to display text. Anything in quotes will be shown on the screen.

Step 3: Variables and Data Types

name = "Alice"
age = 25
height = 5.4
is_student = True
Explanation: Variables store values. name is a string, age is an integer, height is a float, and is_student is a boolean (True/False).

Step 4: Operators

a = 10
b = 3
print(a + b)
print(a - b)
print(a * b)
print(a / b)
print(a % b)
Explanation: These are arithmetic operations. % gives the remainder. Python follows standard math rules for operations.

Step 5: Conditional Statements

age = 18
if age >= 18:
    print("You're an adult.")
else:
    print("You're a minor.")
Explanation: if checks a condition. If it's true, the first block runs. Otherwise, the else block runs. Blocks are defined by indentation.

Step 6: Loops

For Loop

for i in range(5):
    print("Number:", i)
Explanation: range(5) generates numbers from 0 to 4. i takes each value and prints it.

While Loop

count = 0
while count < 5:
    print(count)
    count += 1
Explanation: This loop runs as long as count is less than 5. count += 1 increases the value each time.

Step 7: Lists and Dictionaries

fruits = ["apple", "banana", "cherry"]
print(fruits[0])
Explanation: A list stores multiple items. Indexes start at 0, so fruits[0] accesses the first item. 
person = {
  "name": "Alice",
  "age": 25
}
print(person["name"])
Explanation: A dictionary holds keyâ€“value pairs. Use the key (e.g., "name") to get its value.

Step 8: Functions

def greet(name):
    print("Hello", name)

greet("Alice")
Explanation: def creates a function. name is a parameter. When calling greet("Alice"), it prints a greeting.

Step 9: Error Handling

try:
    x = int(input("Enter a number: "))
    print(10 / x)
except ValueError:
    print("That was not a number!")
except ZeroDivisionError:
    print("Cannot divide by zero.")
Explanation: try runs code that may fail. If a ValueError or ZeroDivisionError happens, the program shows a friendly message instead of crashing.
