üîÅ Unit 5: Loops

for loop

for i in range(5):
    print("Number:", i)

Repeats 5 times. range(5) gives numbers from 0 to 4.

while loop

count = 0
while count < 5:
    print(count)
    count += 1

Repeats while the condition is true. Adds 1 to count each loop.
