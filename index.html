üîÅ Unit 4: Loops

for loop

for i in range(5):
    print("Number:", i)

range(5) gives numbers from 0 to 4. i is each number. Loop repeats 5 times.

while loop

count = 0
while count < 5:
    print(count)
    count += 1

Repeats while count is less than 5. count += 1 adds 1 each time.
