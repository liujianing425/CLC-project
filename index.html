Unit 1 â€“ print()

Definition

print() is a built-in Python function that outputs text or variables to the screen.

How to Use

Write print() followed by the text or variable in parentheses and quotation marks (for text).

Example

print("Hello, world!")

This line prints the text "Hello, world!" to the screen.

Unit 3 â€“ Operators

Definition

Operators perform actions on variables and values, like addition or comparison.

How to Use

Use math operators like +, -, *, / for arithmetic. Use ==, !=, >, < for comparisons.

Example

x = 10
y = 5
print(x + y)       # 15
print(x > y)       # True

Unit 4 â€“ Conditional Statements

Definition

Conditional statements let your program make decisions using if, elif, and else.

How to Use

Use if to check a condition, elif for other conditions, and else if none match.

Example

age = 18
if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")
