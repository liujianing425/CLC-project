This code does math using variables a and b:

  • * multiplies
  • / divides
  • % finds the remainder

üîÄ Step 5: Conditional Statements

age = 18
if age >= 18:
    print("You're an adult.")
else:
    print("You're a minor.")

If-else checks a condition:

  • If the age is 18 or more, it prints "You're an adult."
  • If not, it prints "You're a minor."

Note: Indentation (spaces at the beginning of the line) is required in Python.

üîÅ Step 6: Loops

for loop

for i in range(5):
    print("Number:", i)

range(5) gives numbers from 0 to 4. i is each number. The loop repeats 5 times.

while loop

count = 0
while count < 5:
    print(count)
    count += 1

This repeats as long as count is less than 5. count += 1 means add 1 to count each time.
