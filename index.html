üñ®Ô∏è Unit 1 ‚Äì print() Function

üìå What is print()?

The print() function in Python displays messages, numbers, or variable values in the console. It helps us **see what the program is doing**.

üí° Syntax

print("your message here")

Write print(), put your text or variable inside it in parentheses. Text must be inside **quotation marks** (" ").

üéØ Example

print("Hello, Python!")

This code will show:

Hello, Python!

üìÇ Multiple Items

name = "Alice"
  print("Hello", name)

print() can handle **multiple arguments**, separated by commas. It prints them with spaces in between.

Unit 2 ‚Äì Variables & Data Types

Definition

Variables are containers for storing data values. Data types specify what kind of data a variable holds, such as numbers, text, or true/false values.

How to Use

Use the = sign to assign a value to a variable. Common types are:

  • int ‚Äì whole numbers (e.g., 25)
  • float ‚Äì decimals (e.g., 5.6)
  • str ‚Äì text strings (e.g., "Hello")
  • bool ‚Äì Boolean values (True or False)

Example

  age = 25
  height = 5.6
  is_student = True

Unit 3 ‚Äì Operators

Definition

Operators let you perform actions on values and variables, such as math calculations or comparisons.

How to Use

Python supports:

  • Arithmetic operators: +, -, *, /
  • Comparison operators: ==, !=, >, <

Example

x = 10
  y = 5
  
  print(x + y)    # Output: 15
  print(x > y)    # Output: True
  print(x == y)   # Output: False

Use if to check a condition. If it‚Äôs true, the indented code below it runs. elif is for additional conditions, and else handles everything else.

      print("You are an adult.")
